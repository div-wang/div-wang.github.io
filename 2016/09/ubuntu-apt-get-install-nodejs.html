更新apt-get源的方法有好多，这里推荐nodejs官方的源：nodesource/distributions

介绍一下用法：

1、添加一个NodeSource签名密钥

curl --silent https://deb.nodesource.com/gpgkey/nodesource.gpg.key | sudo apt-key add -

2、在apt下增加一个nodesource.list文件

    cd /etc/apt/sources.list.d/ && vim nodesource.list

3、编辑nodesource.list文件

加入以下两行代码：

    deb https://deb.nodesource.com/node_4.x trusty main
    deb-src https://deb.nodesource.com/node_4.x trusty main

node_4.x是版本号，可以指定安装版本。

4、更新apt-get源&&安装nodejs

    sudo apt-get update
    sudo apt-get install nodejs
