### 如何开始
请修改 `src` 目录里的对应文件

* 本地有 Node.JS 和 Python 2.7 环境;  
* 克隆本项目`https://github.com/div-wang/div-wang.github.io.git`; 
* 进入项目目录执行 `npm install`;   
* 终端输入 `gulp` 即可开始
